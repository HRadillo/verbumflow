# VerbumFlow · Brand Assets

All logo and icon files in SVG (source of truth) and PNG (rasterized).

## Folders

- `mark/` — Primary mark (V + tilde split). 7 variants: primary (transparent), on-light, on-dark, on-blue, on-coral, mono-ink, mono-white. PNGs at 512 / 1024 / 2048 px.
- `icon/` — Rounded-square app/web icon. 4 tile colors: ink, paper, blue, coral. PNGs at 16 / 32 / 48 / 64 / 128 / 256 / 512 / 1024 px.
- `lockup/` — Horizontal lockup (mark + wordmark + tagline). 4 backgrounds: light, dark, blue, coral. PNGs at 1200 / 2400 px.
- `wordmark/` — Wordmark only. 3 variants: light, dark, mono-ink. PNGs at 1200 px.

## Colors

- Electric Blue `#1F4BFF`
- Coral `#FF6A4D`
- Deep Ink `#0B1020`
- Paper `#FAFAF7`

## Notes

- SVGs are the master. Edit/scale from those.
- PNGs are transparent where no background is declared (the "primary" / "mono-ink" marks). Colored variants include a full-bleed background.
- Wordmark PNGs use Plus Jakarta Sans; embed the font if you rebuild from SVG in another tool.
